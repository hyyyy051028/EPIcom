# 培养方向

EPI 软件实验室强调“基础 + 实践 + 协作”的培养方式。

## Web 与产品开发

围绕现代 Web 技术栈开展训练，内容通常包括：

- HTML / CSS / JavaScript / TypeScript
- React、Vue 等前端框架
- Node.js 与接口联调
- 页面性能、组件设计与工程化规范

## 后端与系统能力

帮助成员建立服务端开发与系统设计意识：

- 常见 Web 服务开发流程
- 数据库设计与接口规范
- 缓存、权限、日志等基础能力
- 部署、监控与问题排查

## 工程实践能力

实验室注重把“会写代码”推进到“会做项目”：

- Git 协作与分支管理
- 代码评审与文档沉淀
- 测试意识与质量保障
- 任务拆解、排期与项目复盘

## 团队协作与表达

优秀的工程师不仅要能实现功能，也要能清晰表达和稳定协作：

- 技术分享与知识传递
- 需求沟通与方案讨论
- 组内 mentoring 与阶段反馈
- 面向结果的协作习惯
