# 加入我们

如果你对软件开发有热情，并愿意投入时间持续学习，EPI 软件实验室欢迎你的加入。

## 我们看重的特质

相比“已经掌握很多技术”，我们更看重：

- 对技术有持续好奇心
- 愿意主动学习和查阅资料
- 能够坚持完成阶段任务
- 乐于沟通并接受反馈

## 常见准备建议

你可以提前做这些准备：

- 打好一门语言基础，如 C/C++、Java、Go、JavaScript 等
- 熟悉 Git 与基本命令行操作
- 尝试完成一个自己的小项目
- 关注实验室招新通知与说明

## 参与方式

实验室通常会通过宣讲、笔试、面试或项目考察等形式了解同学的兴趣与能力情况。具体安排请以实验室当年通知为准。

## 想加入后的建议

进入实验室后，最重要的是保持节奏：

- 主动参加例会与分享
- 及时汇报进度与问题
- 在真实项目中锻炼执行力
- 通过复盘不断提升自己的方法论
