# 实验室介绍

西安邮电大学 EPI 软件实验室致力于培养具备工程思维、项目意识与协作能力的软件人才。

## 我们关注什么

实验室以软件工程实践为核心，鼓励成员从课堂知识走向真实项目，从个人编码走向团队协同。

- 以项目驱动学习，重视需求分析、代码质量与交付流程
- 以技术分享促进成长，建立稳定的学习共同体
- 以竞赛、开源、实战项目拓宽成员视野

## 我们希望你获得什么

在这里，你不仅能学习具体技术栈，还会逐步建立完整的软件研发认知：

- 如何拆解一个问题并制定实现方案
- 如何在团队中高效沟通与协作
- 如何通过代码评审、测试与复盘持续改进
- 如何把兴趣沉淀为长期可持续的成长能力

## 面向对象

欢迎对以下方向感兴趣的同学加入：

- 前端与全栈开发
- 后端与系统设计
- 工程化与自动化工具
- 算法竞赛与技术探索

## 实验室文化

我们鼓励开放、主动、务实的交流方式。成员可以提出想法、发起项目、组织分享，也需要对自己的学习节奏和团队合作负责。

继续阅读后续文档，了解培养方向、日常机制与加入方式。
