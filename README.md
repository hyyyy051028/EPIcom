# EPIcom

西安邮电大学 EPI 软件实验室官网项目，基于 VitePress 构建，并通过 Cloudflare 部署。

## 环境配置

- 确保 Node.js 版本为 20.0 或更高版本。

```bash
node -v
npm install -g pnpm
```

## 安装依赖

```bash
pnpm install
```

## 本地开发

```bash
pnpm run dev
```

## 构建

```bash
npm run build
```

## 预览

```bash
npm run preview
```

## 部署

```bash
npm run deploy
```
